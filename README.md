# Requirements
To properly run this project, <br>
1- Run a json server by executing the command: <br>
`json-server --watch dbdisetereta.json`<br>
2- Edit your dbdisetereta.json file to look like this:<br>
`{
  "days": []
}`<br>
ENJOY!
